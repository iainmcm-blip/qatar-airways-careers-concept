# Qatar Airways Careers — Concept

An independent concept for Qatar Airways' careers & employer-brand platform — an
8-page Next.js demo built in the airline's real brand identity (burgundy + gold),
based in Doha and connecting the globe.

**Independent concept by Iain McMullan. Not affiliated with or endorsed by Qatar
Airways. All brand assets remain the property of Qatar Airways Group.**

## Pages

- **Home** — hero, Al Darb (Qatarisation), why Qatar Airways, teams, open roles, stories, benefits, life in Doha
- **Life at Qatar Airways** — culture, rewards, diversity, sustainability, wellbeing, life in Doha
- **Our Teams** — Cabin Crew, Pilots, Engineering, Operations & Corporate, Cargo & Airport, Customer Services
- **Our Stories** — profiles, day-in-the-life, career journeys, nationalities
- **Qatari Nationals** — the Al Darb Qatarisation programme & pathways
- **Your Journey** — the recruitment process, tips, FAQs, fraud guidance
- **Events** — open days, webinars, assessment days
- **Search Jobs** — filterable role browser

## Stack

- Next.js (App Router) · React 19 · TypeScript
- Tailwind CSS v4 (brand tokens in `app/globals.css`)
- Framer Motion (scroll reveals) · hls.js (background video)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Notes on assets

Imagery and the hero video are drawn from Qatar Airways' public careers media for
this concept. Employee stories, illustrative programmes and demo roles are clearly
flagged in-app and would be replaced with verified content in production.
